# React Native + Vite Project 🚀

This project is built with **React Native** and **Vite** for a fast development experience.

---

## 📦 Requirements

- [Node.js](https://nodejs.org/) (LTS recommended)
- npm / yarn / pnpm
- React Native CLI
- Xcode (for iOS) or Android Studio (for Android)

---

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/your-username/your-project.git

cd your-project

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
